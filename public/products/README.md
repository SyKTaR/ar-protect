# Photos produits

Les nouveaux produits utilisent actuellement `/arprotect_logo.jpg` comme visuel par défaut.

Pour afficher de vraies photos produit, place les fichiers dans ce dossier avec ces noms :

- `shampoing-ph-neutre.jpg`
- `quick-detailer.jpg`
- `nettoyant-jantes.jpg`
- `nettoyant-interieur.jpg`
- `dressing-pneus.jpg`
- `soin-cuirs-nourrissant.jpg`
- `protection-vitres-hydrophobe.jpg`
- `pack-entretien-complet.jpg`

Formats possibles : `.jpg`, `.png`, `.webp`. Si tu changes l'extension, pense aussi a modifier le champ `image.src` dans `lib/products.ts`.
